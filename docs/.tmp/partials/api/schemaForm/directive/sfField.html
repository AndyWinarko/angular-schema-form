sfField

  1. - directive in module schemaForm

sf-field directive keeps track and exports the canonical form definition object for use in the template. Usually added to the first element by the sfField builder function.

sf-field exports a ngModelController on scope under the name ngModel, if a schema-validate directive is used inside the template.

It also exports a couple of useful functions on scope.

Directive Info

  • This directive executes at priority level 0.

Usage

<div sf-field="0">{{form.title}}</div>

Arguments

Param Type Details
sfField number

form definition id.
