schemaForm

Generate forms from a JSON-Schema.

Installation

First include angular-schema-form javascript file in your HTML:

You can download this file from the following places:

  • Bower
    e.g. 
    bower install angular angular-schema-form

Then load the module in your application by adding it as a dependent module:

angular.module('app', ['schemaForm']);

With that you're ready to get started!

Module Components

Directive

Name Description
sfField

sf-field directive keeps track and exports the canonical form definition object for use in the template. Usually added to the first element by the sfField builder function.
